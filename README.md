## Property Management System (Python)

This is a console-based Property Management System developed using Python. 
The project allows sellers to add, update, and delete property records, and buyers to search, view, and book properties.

### Features
- Seller login to add land or residential properties
- Buyer login to search properties by:
  - Location
  - Price range
  - Property type
- Property booking system
- Binary file handling using `pickle`

### Modules
- `seller.py` – Add, delete, and update property details
- `buyer.py` – Search, display, and book properties
- `main.py` – Login system and menu-driven integration

### Data Storage
- Uses binary `.dat` files for storing property records

### Tools Used
- Python
- Pickle module
